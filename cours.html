Fiches de cours — Loi normale

Tous les cas de figure que tu dois connaître pour l'examen

Les deux formules à connaître par cœur

FORMULE 1 — Transformer x en z
z = (x − m) / s
FORMULE 2 — Transformer z en x
x = (z × s) + m

m = moyenne  |  s = écart-type

1 Table directe — Trouver P sachant z

La table donne toujours P(Z < z) pour z ≥ 0. On utilise z = 1.25 partout pour voir les liens.

Cas 1P(Z < z) avec z positif lecture directe

Le plus simple : tu lis directement dans la table. L'aire bleue = probabilité cherchée.

P(Z < 1.25) → lecture directe → 0.8944
Aire bleue = P(Z < 1.25) = 0.8944
Cas 2P(Z > z) avec z positif 1 − P(Z < z)

On veut l'aire à droite. Comme l'aire totale = 1 :

P(Z > 1.25) = 1 − P(Z < 1.25) = 1 − 0.8944 = 0.1056
Aire bleue = P(Z < 1.25) = 0.8944
Aire rouge = 1 − 0.8944 = 0.1056

Bleue + rouge = 1 (toute l'aire sous la courbe)

Cas 3P(Z < −z) z négatif = P(Z > z) = 1 − P(Z < z)

Pas de z négatif dans la table ! On utilise la symétrie : l'aire à gauche de −1.25 = l'aire à droite de +1.25.

P(Z < −1.25) = P(Z > 1.25) = 1 − 0.8944 = 0.1056
P(Z < −1.25) — aire à gauche
P(Z > 1.25) — même aire par symétrie !

↔ Les deux aires rouges sont identiques (symétrie)

Cas 4P(Z > −z) = P(Z < z) → lecture directe

Symétrie encore : l'aire à droite de −1.25 = l'aire à gauche de +1.25.

P(Z > −1.25) = P(Z < 1.25) = 0.8944
P(Z > −1.25) — aire à droite
P(Z < 1.25) — même aire !
Cas 5P(z1 < Z < z2) encadrement P(Z < z2) − P(Z < z1)

On prend l'aire à gauche de z2, puis on retire l'aire à gauche de z1 :

P(0.50 < Z < 1.25) = P(Z < 1.25) − P(Z < 0.50) = 0.8944 − 0.6915 = 0.2029

2 Table inverse — Trouver z (ou x) sachant P

On connaît l'aire (la probabilité) et on cherche où placer la ligne verticale. On utilise P = 0.75 et P = 0.25 pour montrer les liens.

Cas AP(Z < z) = P > 0.5lecture directe

P > 0.5 → z est positif (à droite de 0).

P(Z < z) = 0.75 → table inverse → z = 0.6745
On connaît l'aire (0.75) → z = 0.67
Cas BP(Z < z) = P < 0.5lecture puis −z

P < 0.5 → z est négatif (à gauche de 0). On lit dans la table puis on met −.

P(Z < z) = 0.25 → table : 0.6745 → z = −0.6745
Aire = 0.25 à gauche → z est négatif = −0.67
Cas CP(Z > z) = P > 0.5lecture puis −z

75% au-dessus de z → z doit être à gauche de 0 → négatif.

P(Z > z) = 0.75 → table : 0.6745 → z = −0.6745
75% au-dessus → z est à gauche = −0.67
Cas DP(Z > z) = P < 0.5z positif (2 changements s'annulent)

On changerait le signe 2 fois → ça s'annule. z reste positif.

P(Z > z) = 0.25 → table : 0.6745 → z = 0.6745
25% au-dessus → z est à droite = 0.67

N'oublie pas de retransformer z en x !

Si l'énoncé demande x (pas z) → formule 2 :

x = (z × s) + m

3 Valeurs atypiques

|z| > 2 → valeur atypique

~4.6% des valeurs tombent dans les zones rouges (2.3% de chaque côté).

Zones rouges : |z| > 2 → valeurs atypiques

4 Propriété de symétrie

P(Z < −z) = P(Z > z)  |  P(Z > −z) = P(Z < z)

Exemple : QI ~ N(100, 15). P(100 < X < 110) = P(90 < X < 100) car 110 et 90 sont à 10 points de la moyenne.

P(100 < X < 110) ≈ 24.86%
P(90 < X < 100) ≈ 24.86%

↔ Même écart à la moyenne = même probabilité !
