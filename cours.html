Fiches de cours — Loi normale

Tous les cas de figure que tu dois connaître pour l'examen

Les deux formules à connaître par cœur

FORMULE 1 — Transformer x en z
z = (x − m) / s
FORMULE 2 — Transformer z en x
x = (z × s) + m

m = moyenne | s = écart-type

🔔 Pourquoi la loi normale ?

En psychologie, beaucoup de variables mesurées (eg., le QI, les scores d'anxiété, les temps de réaction, la durée de sommeil…) se répartissent selon une courbe en cloche : la majorité des individus se concentre autour de la moyenne, et plus on s'en éloigne, plus les valeurs deviennent rares. C'est cette forme régulière qu'on appelle la loi normale.

Grâce à cette propriété, il suffit de connaître la moyenne (m) et l'écart-type (s) d'une distribution pour pouvoir répondre à des questions très concrètes : « Quel pourcentage de la population dépasse tel score ? », « En dessous de quelle valeur se trouvent les 10 % les plus faibles ? », « Ce résultat est-il atypique ? ». C'est exactement ce que nous avons appris à faire avec les tables de la loi normale.

Exemple : QI ~ N(100, 15) — ~68 % des personnes entre 85 et 115, ~95 % entre 70 et 130

1 Table directe — Trouver P sachant z ▼

La table donne toujours P(Z < z) pour z ≥ 0. On utilise z = 1.25 partout.

Cas 1P(Z < z) avec z positif lecture directe

Le plus simple : tu lis directement dans la table.

P(Z < 1.25) → lecture directe → 0.8944
Aire bleue = P(Z < 1.25) = 0.8944
Cas 2P(Z > z) avec z positif 1 − P(Z < z)

On veut l'aire à droite. Comme l'aire totale = 1 :

P(Z > 1.25) = 1 − P(Z < 1.25) = 1 − 0.8944 = 0.1056
P(Z < 1.25) = 0.8944
1 − 0.8944 = 0.1056

Bleue + rouge = 1

Cas 3P(Z < −z) z négatif = P(Z > z) = 1 − P(Z < z)

Pas de z négatif dans la table ! On utilise la symétrie.

P(Z < −1.25) = P(Z > 1.25) = 1 − 0.8944 = 0.1056

2 Table inverse — Trouver z (ou x) sachant P ▼

On connaît l'aire (la probabilité) et on cherche où placer la ligne verticale.

Cas AP(Z < z) = P > 0.5lecture directe

P > 0.5 → z est positif.

P(Z < z) = 0.75 → table inverse → z = 0.6745
Aire (0.75) → z = 0.67
Cas BP(Z < z) = P < 0.5lecture puis −z

P < 0.5 → z est négatif. On lit dans la table puis on met −.

P(Z < z) = 0.25 → table : 0.6745 → z = −0.6745
Aire = 0.25 → z négatif = −0.67

75% au-dessus de z → z à gauche de 0 → négatif.

P(Z > z) = 0.75 → table : 0.6745 → z = −0.6745
75% au-dessus → z = −0.67
Cas DP(Z > z) = P < 0.5z positif

On changerait le signe 2 fois → ça s'annule. z reste positif.

P(Z > z) = 0.25 → table : 0.6745 → z = 0.6745
25% au-dessus → z = 0.67

N'oublie pas de retransformer z en x !

Si l'énoncé demande x (pas z) → formule 2 :

x = (z × s) + m

3 Valeurs atypiques ▼

|z| > 2 → valeur atypique

~4.6% des valeurs tombent dans les zones rouges (2.3% de chaque côté).

Zones rouges : |z| > 2 → valeurs atypiques

4 Propriété de symétrie ▼

P(Z < −z) = P(Z > z)  |  P(Z > −z) = P(Z < z)

Exemple : QI ~ N(100, 15). P(100 < X < 110) = P(90 < X < 100) car 110 et 90 sont à 10 points de la moyenne.

P(100 < X < 110) ≈ 24.86%
P(90 < X < 100) ≈ 24.86%

↔ Même écart à la moyenne = même probabilité !
