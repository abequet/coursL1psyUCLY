Exercices — Lecture de table

Utilise ta table papier puis vérifie avec le corrigé illustré !

Rappel : la table donne P(Z < z) pour z ≥ 0.  |  P(Z > z) = 1 − P(Z < z)  |  P(Z < −z) = 1 − P(Z < z)
1 Table directe Fondamental

Utilise ta table papier, écris ta réponse, puis clique « Valider » pour vérifier.

a) P(Z < 0.85) = ?
Règle
P(Z < z) positif → lecture directe
Ligne 0.8, colonne 0.05
P(Z < 0.85) = 0.8023
Aire bleue = 0.8023
b) P(Z < 2.17) = ?
P(Z < 2.17) → lecture directe → ligne 2.1, colonne 0.07
= 0.9850
c) P(Z > 1.54) = ?
Règle
P(Z > z) → 1 − P(Z < z)
= 1 − P(Z < 1.54) = 1 − 0.9382 = 0.0618
= 0.0618
Aire rouge = 0.0618
d) P(Z > 0.73) = ?
P(Z > 0.73) = 1 − P(Z < 0.73) = 1 − 0.7673
= 0.2327
e) P(Z < −0.91) = ?
Règle
P(Z < −z) = P(Z > z) → 1 − P(Z < z)
= 1 − P(Z < 0.91) = 1 − 0.8186
= 0.1814
Aire rouge à gauche de −0.91
f) P(Z < −2.05) = ?
P(Z < −2.05) = 1 − P(Z < 2.05) = 1 − 0.9798
= 0.0202
g) P(Z > −1.67) = ?
Règle
P(Z > −z) = P(Z < z) → lecture directe
= P(Z < 1.67)
= 0.9525
Aire bleue à droite de −1.67
h) P(Z > −2.41) = ?
P(Z > −2.41) = P(Z < 2.41)
= 0.9920
i) P(0.50 < Z < 1.96) = ?
Règle
Encadrement → P(Z < z2) − P(Z < z1)
= P(Z < 1.96) − P(Z < 0.50) = 0.9750 − 0.6915
= 0.2835
Aire verte entre 0.50 et 1.96
j) P(−1.50 < Z < 0.80) = ?
= P(Z < 0.80) − P(Z < −1.50)
P(Z < −1.50) = 1 − P(Z < 1.50) = 1 − 0.9332 = 0.0668
= 0.7881 − 0.0668
= 0.7213

Table : 1.0364 → z = −1.0364

P(Z > z) et P < 0.5 → 2 changements s'annulent → z positif

e) X ~ N(40 ; 8). x ? sachant P(X < x) = 0.90
Étape 1
P(Z < z) = 0.90 → z = 1.2816
Étape 2 — Formule 2
x = (1.2816 × 8) + 40 = 10.25 + 40
x = 50.25
